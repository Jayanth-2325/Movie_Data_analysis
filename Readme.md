# Movie Dataset Analysis

## Project Overview

This project focuses on analyzing a movie dataset to discover patterns in ratings, genres, directors, movie releases, and revenue.
The analysis is performed using Python with data visualization techniques to understand trends and insights.

---

## Tools & Technologies

* Python
* Pandas
* Matplotlib

---

## Project Files

* `datacleaning.ipynb` → Data cleaning and preprocessing
* `Director_analysis.py` → Director rating analysis
* `Movie_by_year.py` → Movies released per year
* `Movie_Gross_analysis.py` → Highest gross movies
* `Top_Movies_genre_distribution.py` → Genre distribution
* `Movie_dataset.csv` → Dataset

---

##Analysis & Visualizations

###Average Rating by Genre

* Displays average ratings across different genres
* Insight: Most genres have similar ratings (~8), indicating consistent movie quality

---

###Director Rating Analysis

* Shows top directors based on average movie ratings
* Insight: Top directors consistently produce highly rated movies

---

###Highest Gross Movies

* Identifies top 10 movies based on total gross revenue
* Gross values converted into million USD for better readability
* Insight: A few blockbuster movies generate significantly higher revenue

---

###Genre Distribution

* Shows top 5 genres using a pie chart
* Insight: Drama is the most dominant genre in the dataset

---

###Movies Released Per Year

* Displays number of movies released each year
* Insight: Movie production varies by year with some peak periods

---

##Key Insights

* Most movies have ratings between 6 and 8
* Drama and Action genres are most common
* Popular movies tend to receive higher audience engagement
* Revenue is concentrated among a few top movies
* Director quality plays an important role in movie success



##Conclusion

The project successfully identifies patterns in movie ratings, genres, and revenue.
Data visualization helps in understanding trends and relationships effectively.
